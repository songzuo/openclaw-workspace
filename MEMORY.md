## 🧠 记忆系统 (mem9风格)

### 使用方法

```bash
# 存储记忆
python3 scripts/memory-manager.py store "要记住的内容"

# 搜索记忆
python3 scripts/memory-manager.py search "关键词"

# 查看列表
python3 scripts/memory-manager.py list

# 获取单条
python3 scripts/memory-manager.py get <key>
```

### 索引文件
- `memory/index.json` - 关键词索引
- `memory/YYYY-MM-DD.md` - 每日记忆

### 设计理念 (来自 mem9)
- 自动关键词索引（支持中英文）
- 模糊搜索（包含匹配）
- 生命周期管理（自动存储）
- 云端同步（通过 GitHub）

## 💡 教训总结

### 教训1: 配置守护
- 问题: doctor会覆盖配置文件
- 解决: 配置守护进程，或用环境变量

### 教训2: 公众号文章不重复
- 问题: 写公众号文章用了旧内容
- 解决: 每天生成全新内容，阅读当天资讯

### 教训3: 记忆混乱 (2026-03-15)
- 问题: 两个公众号配置找不到
- 原因: 配置写在config.json但没同步到MEMORY.md
- 解决: 新配置必须立刻写入MEMORY.md

### 教训4: 搜索能力
- 问题: 只搜关键词搜不到
- 解决: 用多种关键词组合（名字+ID+时间）
