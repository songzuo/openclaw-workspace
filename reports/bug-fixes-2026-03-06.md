#### 🟡 中等 Bug #3: 内存数据库初始化竞态条件

**位置:** `app/lib/users.ts:26-37`

```typescript
async function initUsers() {
  const hashedPassword = await bcrypt.hash('admin123', 10);
  users[0].password = hashedPassword;
  // ...
}

initUsers(); // ⚠️ 没有 await，异步初始化未等待
```

**问题:**
- `initUsers()` 是异步函数但未等待
- 模块加载时立即调用，但密码哈希可能未完成
- 并发请求可能导致密码验证失败

**影响:** 🟡 中等 - 登录可能间歇性失败

**修复建议:**
```typescript
// 使用 IIFE 并导出 Promise
export const usersInitialized = (async () => {
  const hashedPassword = await bcrypt.hash('admin123', 10);
  users[0].password = hashedPassword;
  // ...
})();

// 在 API 路由中等待初始化
export async function findUserByEmail(email: string) {
  await usersInitialized;
  return users.find(u => u.email === email);
}
```

#### 🟡 中等 Bug #4: 静默时段跨午夜逻辑错误

**位置:** `app/lib/notifications/utils.ts:69-82`

```typescript
export function isInQuietHours(start: string, end: string): boolean {
  const now = new Date();
  const currentTime = now.getHours() * 60 + now.getMinutes();
  
  const [startHour, startMin] = start.split(':').map(Number);
  const [endHour, endMin] = end.split(':').map(Number);
  
  const startTime = startHour * 60 + startMin;
  const endTime = endHour * 60 + endMin;
  
  if (startTime <= endTime) {
    return currentTime >= startTime && currentTime < endTime;
  } else {
    // 跨午夜的情况
    return currentTime >= startTime || currentTime < endTime;
  }
}
```

**问题:**
- 跨午夜逻辑正确，但测试用例未覆盖边界情况
- 当 `start='22:00'`, `end='08:00'` 时:
  - 23:00 ✅ 应返回 true
  - 07:00 ✅ 应返回 true
  - 08:00 ❌ 应返回 false (但代码返回 true，因为 `<` 而非 `<=`)

**影响:** 🟡 中等 - 安静时段边界时间通知行为不一致

**修复建议:**
```typescript
// 明确处理边界
return currentTime >= startTime || currentTime <= endTime;
```

#### 🟡 中等 Bug #5: CSV 导出未处理换行符

**位置:** `app/lib/export.ts:23-32`

```typescript
function arrayToCSV(data: Record<string, unknown>[]): string {
  // ...
  ...data.map(row =>
    headers.map(header => {
      const value = row[header];
      // 处理包含逗号或引号的值
      if (typeof value === 'string' && (value.includes(',') || value.includes('"'))) {
        return `"${value.replace(/"/g, '""')}"`;
      }
      return value ?? '';
    }).join(',')
  )
}
```

**问题:**
- 未处理字符串中的换行符 (`\n`, `\r`)
- 包含换行符的值会破坏 CSV 格式
- 可能导致导出文件无法正确解析

**影响:** 🟡 中等 - CSV 导出可能损坏

**修复建议:**
```typescript
if (typeof value === 'string' && 
    (value.includes(',') || value.includes('"') || value.includes('\n') || value.includes('\r'))) {
  return `"${value.replace(/"/g, '""')}"`;
}
```

export function formatNotificationTime(timestamp: number): string {
  const now = Date.now();
  const diff = now - timestamp;
  // ⚠️ 如果 timestamp > now (未来时间),diff 为负数
  const seconds = Math.floor(diff / 1000);
  // ...
}
