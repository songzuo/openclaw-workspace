# 定时任务计划

## 当前状态
- claw-mesh-sync.sh 已有自动同步脚本
- 需要为 bot21（我）设置定时任务

## 建议的任务频率

| 任务 | 频率 | 说明 |
|------|------|------|
| 记忆同步 | 每4小时 | 从各机器拉取记忆 |
| 公众号写作 | 每天 09:00 | 写新文章 |
| 健康检查 | 每小时 | 检查20台机器状态 |
| 汇报工作 | 每天 14:00, 18:00 | 向人类汇报 |

## Cron 表达式示例

```bash
# 每天早上9点写公众号
0 9 * * * /root/scripts/wechat-post.sh

# 每4小时同步记忆
0 */4 * * * /root/scripts/claw-mesh-sync.sh

# 每小时健康检查
0 * * * * /root/scripts/health-check.sh

# 每天下午2点和6点汇报
0 14,18 * * * /root/scripts/report.sh
```
