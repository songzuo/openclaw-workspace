## 📋 项目概述

### 7zi Studio 介绍

7zi Studio 是一个由 AI 驱动的创新工作室，专注于开发智能化工具和解决方案。我们致力于将人工智能技术融入日常工作流程，提升效率和创造力。

**使命**：让 AI 成为每个人的得力助手，而非替代品。

**愿景**：构建一个人机协作的未来，其中 AI 处理重复性工作，人类专注于创造性决策。

### 部署脚本

```bash
#!/bin/bash
# 7zi Studio 快速部署脚本

# 环境检查
echo "🔍 检查环境..."
node --version
npm --version

# 安装依赖
echo "📦 安装依赖..."
npm install

# 配置验证
echo "⚙️ 验证配置..."
openclaw config validate

# 启动服务
echo "🚀 启动服务..."
openclaw gateway start

echo "✅ 部署完成!"
```

### 配置示例

```yaml
# .openclaw/config.yaml
gateway:
  port: 8080
  host: localhost
  
models:
  default: minimax/MiniMax-M2.5
  fallback: bailian/qwen3.5-plus
  
skills:
  enabled:
    - coding-agent
    - healthcheck
    - weather
    - team-meeting
    
workspace:
  path: /root/.openclaw/workspace
  auto_backup: true
```
